# MyNews
各国のトップニュースを見ることができ、希望する記事を検索しても閲覧できる、誰でも使える簡単なニュースアプリです。
## Feature
* １ページには国家別のトップニュースが見えます。
* ２ページには検索して記事を探して見えます。
* ３ページにはニュースの出所が出ます。
* 詳細な記事はURLに移動して見えます。
* News APIを利用しています。
